# Аттестация ПМ.02
Проект представляет собой консольное приложение на C#, моделирующее работу археологической экспедиции в условиях пустыни. Симулятор демонстрирует принципы объектно-ориентированного программирования, обработку исключений и работу с доменной логикой.

# О проекте

В симуляторе представлена команда археологов, которая:

Проводит раскопки с использованием специальных инструментов

Управляет исследовательским дроном для разведки местности

Планирует маршруты с учетом погодных условий и запасов воды

Обнаруживает и классифицирует артефакты

# Основные возможности

Роли персонажей
Археолог - проводит раскопки, находит артефакты

Инженер - управляет и ремонтирует исследовательский дрон

Проводник - планирует маршруты, управляет запасами воды

# Система артефактов

Типы артефактов: Керамика, Инструменты, Таблички, Украшения

Учет находок: вес, координаты обнаружения, классификация

Инвентаризация: вечерняя сводка по найденным артефактам

# Погодные условия

Ясно - оптимальные условия для работы

Ветрено - повышенный риск поломки оборудования

Песчаная буря - приостановка большинства работ

# Оборудование

Инструменты для раскопок - имеют ограниченную прочность

Исследовательский дрон - может выходить из строя в плохую погоду

Система ремонта - инженер может восстановить работоспособность дрона

# Архитектура

Основные компоненты
Entities - базовые сущности системы (персонажи, оборудование)

Characters - игровые персонажи с специализированным поведением

Items - инвентарь (инструменты, артефакты)

World - игровой мир с состоянием погоды и коллекцией артефактов

ScenarioEngine - движок сценария, управляющий ходом simulation

Ключевые интерфейсы
ILogger - система логирования

IInventoryItem - предметы инвентаря

IRepairable - ремонтируемые объекты

# Обработка ошибок

DomainCheckedException - базовое доменное исключение

NotEnoughSuppliesException - недостаток ресурсов

ToolBrokenException - поломка инструмента

# Запуск и использование

Требования
.NET 5.0 или выше

Поддержка C# 9.0 (records)

Сборка и запуск
bash
dotnet build
dotnet run
